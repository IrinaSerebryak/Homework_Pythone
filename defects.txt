## КРИТИЧЕСКИЕ ДЕФЕКТЫ:

### 1. Метод: is_empty()
**Проблема:** Неправильная обработка строки, состоящей только из пробелов
**Ожидаемое поведение:** Метод должен возвращать True только для полностью пустой строки ("")
**Фактическое поведение:** Метод возвращает True для строки из пробелов (" ")
**Шаги воспроизведения:**
  1. Вызвать utils.is_empty(" ")
  2. Получить результат True
**Ожидаемый результат:** False
**Фактический результат:** True
**Серьезность:** Высокая

### 2. Метод: list_to_string()
**Проблема:** Неправильная обработка пустого списка с пользовательским разделителем
**Ожидаемое поведение:** Для пустого списка всегда возвращать пустую строку
**Фактическое поведение:** Метод работает корректно только с разделителем по умолчанию
**Шаги воспроизведения:**
  1. Вызвать utils.list_to_string([], "-")
  2. Получить результат ""
**Ожидаемый результат:** ""
**Фактический результат:** "" (работает корректно, но требует проверки)
**Серьезность:** Средняя

## НЕКРИТИЧЕСКИЕ ДЕФЕКТЫ:

### 3. Метод: capitilize()
**Проблема:** Опечатка в названии метода
**Ожидаемое поведение:** Название метода должно быть "capitalize"
**Фактическое поведение:** Метод называется "capitilize"
**Серьезность:** Низкая

### 4. Метод: trim()
**Проблема:** Удаляются только пробелы в начале строки, но не другие whitespace-символы
**Ожидаемое поведение:** Должны удаляться все whitespace-символы (табы, переносы и т.д.)
**Фактическое поведение:** Удаляются только пробелы
**Серьезность:** Низкая
